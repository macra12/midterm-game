# Vehicle Obstacle Dodger - GAD Midterm Project

## Overview
This is a 3D driving and obstacle avoidance game created in Unity. The player controls a vehicle that must navigate and survive by dodging incoming obstacles. 

This project was developed as a midterm assignment for the Game Application Development (GAD) course during Term 2, Year 4 at the Royal University of Phnom Penh (RUPP).

## Gameplay & Controls
The objective is to avoid the flying obstacles for as long as possible. The camera features smooth-follow mechanics for a dynamic viewing angle.

* **Drive Forward:** `W` or `Up Arrow`
* **Reverse/Brake:** `S` or `Down Arrow`
* **Steer Left:** `A` or `Left Arrow`
* **Steer Right:** `D` or `Right Arrow`

## Built With
* **Game Engine:** Unity 3D
* **Language:** C#
* **Physics:** Unity Rigidbody & Box Colliders

## How to Run the Project
1. Clone this repository to your local machine:
   `git clone https://github.com/macra12/midterm-game.git`
2. Open **Unity Hub** and click **Open Project**.
3. Navigate to the cloned folder and select it.
4. Once Unity loads, open the main game scene in the `Assets/Scenes` folder.
5. Press the **Play** button at the top of the Unity Editor to start driving!
